# VibeCare
VibeCare is a proactive digital mental health monitoring platform designed to detect emotional distress before it becomes a crisis. Users complete daily mood check-ins and self-assessments, while the backend analyzes emotional patterns to classify risk levels and deliver personalized coping strategies. The platform includes an AI companion with SOS integration, breathing exercises, journaling, gratitude practice, sport recommendations, and mood trend graphs — all built on a React + FastAPI + MySQL stack.
